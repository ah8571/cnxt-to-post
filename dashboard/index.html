API Reference

Use the cnxt-to-post API from your own apps.

Authentication

All API requests require a Supabase JWT in the Authorization header.

Authorization: Bearer <supabase-access-token>

POST /api/post

Publish to one or more platforms simultaneously.

{ "platforms": ["bluesky", "linkedin"], "text": "Hello world!", "mediaUrls": ["https://example.com/photo.jpg"], "replyTo": "tweet-id-here" }

GET /api/metrics/:platform/:postId

Retrieve engagement metrics for a post.

GET /api/metrics/bluesky/abc123

Supported platforms
